# Telegram AI Bot

Bu proje, Telegram üzerinden günlük soruları cevaplayan yapay zeka destekli basit bir bottur.

## Kullanılan Teknolojiler

- Python
- pyTelegramBotAPI
- OpenRouter API
- python-dotenv
- requests

## Özellikler

- Telegram üzerinden mesaj alır.
- OpenRouter API ile ücretsiz LLM modelinden cevap alır.
- Kullanıcının yazdığı dile göre cevap vermeye çalışır.
- Şimdilik günlük soruları cevaplayacak şekilde tasarlanmıştır.
